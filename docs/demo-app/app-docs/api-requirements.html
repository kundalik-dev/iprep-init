iPrep Current API Requirements

This page matches the current demo-app-2/app.js. It replaces the existing MockAPI methods and direct in-memory edits with real endpoints. It does not include future auth, billing, WebSocket, or real audio APIs.

The current app boots from mock-data.json. The simplest backend replacement is GET /api/bootstrap, returning the same top-level data shape.

Priority Order

  1. GET /api/bootstrap
  2. GET /api/sessions and GET /api/stats
  3. GET /api/packages, GET /api/tutors, session start/end
  4. GET /api/analysis/{id}
  5. Chat endpoints
  6. File-system endpoints
  7. Communication endpoints
  8. Provider and settings endpoints

Not Needed By Current App

Login/logout, billing, pro subscription enforcement, real audio streaming, WebSocket, real speech-to-text, separate analysis job status, and separate tutor/package detail APIs are not required by the current demo-app-2/app.js.
